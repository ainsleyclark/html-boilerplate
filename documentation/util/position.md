# Position
mesh has responsive position classes! You can have something relative for mobile, but absolute for desktop making, like so:

| Notation                |                                                       |          
| ----------------------- | ----------------------------------------------------- | 
| Default                 | p-{position}                                          |   
| Responsive              | p-{breakpoint}-{position} for mob, tab, desk & hd     | 

- ```.p-relative```
- ```.p-absolute```
- ```.p-fixed```
- ```.p-static```
- ```.p-sticky```

And for responsiveness: 

- ```.p-mob-relative```
- ```.p-desk-absolute```
- ```.p-tab-fixed```
- ```.p-hd-static```
- ```.p-tab-sticky```
